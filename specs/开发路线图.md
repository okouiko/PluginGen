# 项目开发路线图 — PluginGen

## 0. 策略概览

- **开发策略**：自底向上，先搭地基后建业务。每一个里程碑都产出一个**可用的线上闭环**——MVP 交付「AI 生成 → 在线查看 → 编译下载」的端到端链路；功能完整版加入社区社交互动；增强版打磨体验与运维。
- **MVP 目标**：用户注册登录后，填写表单 → AI 生成插件 → 在线查看源码 → 沙箱编译 → 下载 JAR/ZIP。**不依赖其他用户**即可完成全部操作。
- **模块依赖关系图**：

```
Layer 0  基础设施 (Scaffolding / Prisma / Docker / Nginx)
   ↓
Layer 1  用户基础 (Auth → User)
   ↓
Layer 2  核心业务 (Plugin → AI Generator → File → Compile)
   ↓
Layer 3  社区社交 (Community → Notification → Daily)
   ↓
Layer 4  增强优化 (Admin / Perf / Polish)
```

---
